## 🎮 使用指南

**🚀 快速开始:** 直接在电脑或手机浏览器中访问 **[在线演示](https://aura.tanox.net/)** 即可，即点即用。

1.  **授权权限:** 点击“开启体验”，允许浏览器访问麦克风。
2.  **播放音乐:** 在设备附近播放音乐。视觉效果将根据实时采样跳动。
3.  **探索模式:** 打开**设置面板** (按 `H`) 切换 12+ 种视觉引擎。
4.  **AI 识别:** 按 `L` 键开启“AI 曲目识别”以分析当前歌曲。

### ⌨️ 快捷键列表

| 按键 | 功能 |
| :--- | :--- |
| **Space (空格)** | 开启/关闭 麦克风 |
| **F** | 切换全屏模式 |
| **R** | 随机切换视觉风格 |
| **L** | 显示/隐藏 AI 识别信息 |
| **H** | 展开/收起 控制面板 |
| **G / T** | 开关 泛光 / 拖尾 |
