## 概述

SonicVision 是一个基于 AI 的音乐可视化应用，能够实时分析音频输入，识别正在播放的歌曲，并根据音乐节奏和特征生成动态的视觉效果。

## 快速开始

1. **访问应用**：打开浏览器，访问 SonicVision 应用 URL
2. **授权麦克风**：点击「开始体验」按钮，授权应用访问麦克风
3. **开始体验**：应用开始捕获音频并生成可视化效果
4. **调整设置**：使用控制面板调整可视化参数和模式

## 核心功能

### 音频捕获与分析

- **实时音频分析**：应用实时分析麦克风输入的音频频率数据
- **设备选择**：支持选择不同的音频输入设备
- **自动检测**：自动检测音乐开始和结束，触发歌曲识别

### 歌曲识别

- **AI 驱动**：使用 Google Gemini API 进行歌曲识别
- **实时识别**：定期录制音频片段并进行识别
- **信息显示**：显示识别结果，包括歌曲标题、艺术家和歌词片段
- **搜索链接**：提供歌曲搜索链接，方便用户了解更多信息

### 可视化效果

#### 2D 可视化模式

- **BARS**：频谱柱状图，显示不同频率的音量
- **PLASMA**：等离子体效果，根据音频数据变形
- **PARTICLES**：粒子效果，粒子随音乐运动
- **TUNNEL**：隧道效果，随音乐节奏移动
- **SHAPES**：几何形状效果，形状随音乐变化
- **RINGS**：环形效果，同心环随音乐脉动
- **NEBULA**：星云效果，类似烟雾的动态效果
- **KALEIDOSCOPE**：万花筒效果，对称图案随音乐变化

#### 3D 可视化模式

- **SILK**：丝绸效果，流动的 3D 表面
- **LIQUID**：液体效果，波动的 3D 表面
- **TERRAIN**：地形效果，起伏的 3D 地形

### 用户控制面板

- **模式切换**：切换不同的可视化模式
- **参数调整**：调整可视化参数，如灵敏度、速度、发光效果等
- **颜色主题**：选择不同的颜色主题
- **歌词控制**：控制歌词显示和样式
- **语言切换**：切换界面语言
- **设置重置**：重置或随机化设置

### 多语言支持

- **语言选择**：支持英语和中文界面
- **自动检测**：根据用户浏览器设置自动选择语言
- **手动切换**：允许用户手动切换语言

## 使用教程

### 首次使用

1. **打开应用**：在浏览器中访问 SonicVision 应用
2. **欢迎界面**：应用显示欢迎界面，介绍核心功能
3. **授权麦克风**：点击「开始体验」按钮，浏览器会弹出麦克风权限请求
4. **允许权限**：点击「允许」按钮，授权应用访问麦克风
5. **开始分析**：应用开始捕获音频并生成可视化效果
6. **等待识别**：应用会自动识别正在播放的歌曲

### 调整可视化参数

1. **打开控制面板**：点击屏幕底部的控制面板图标
2. **调整灵敏度**：使用「灵敏度」滑块调整可视化对音频的敏感度
3. **调整速度**：使用「速度」滑块调整可视化效果的速度
4. **启用/禁用发光**：使用「发光」开关控制是否启用发光效果
5. **启用/禁用轨迹**：使用「轨迹」开关控制是否启用轨迹效果
6. **启用/禁用自动切换**：使用「自动切换」开关控制是否自动切换可视化模式
7. **调整切换间隔**：如果启用了自动切换，使用「切换间隔」滑块调整切换频率

### 切换可视化模式

1. **打开控制面板**：点击屏幕底部的控制面板图标
2. **选择模式类型**：选择「2D 模式」或「3D 模式」标签
3. **选择具体模式**：从模式列表中选择您喜欢的可视化模式
4. **查看效果**：应用会立即切换到所选模式并显示效果

### 使用歌词功能

1. **打开控制面板**：点击屏幕底部的控制面板图标
2. **启用歌词**：确保「显示歌词」开关处于开启状态
3. **选择歌词样式**：从「歌词样式」下拉菜单中选择您喜欢的样式
   - **STANDARD**：标准歌词显示
   - **KARAOKE**：卡拉 OK 样式，突出显示当前歌词
   - **MINIMAL**：极简歌词显示
4. **查看歌词**：当应用识别到歌曲后，会显示歌词

### 切换语言

1. **打开控制面板**：点击屏幕底部的控制面板图标
2. **选择语言**：从「语言」下拉菜单中选择您喜欢的语言
   - **中文**：中文界面
   - **English**：英文界面
3. **应用更改**：界面语言会立即切换

## 常见问题

### Q: 应用无法访问麦克风怎么办？
A: 请检查浏览器设置，确保已授予应用麦克风访问权限。如果使用的是 Chrome 浏览器，可以在地址栏左侧的锁图标中检查和修改权限设置。

### Q: 应用无法识别歌曲怎么办？
A: 请确保：
1. 音乐音量足够大
2. 环境噪音较小
3. 音乐正在播放
4. 网络连接正常

如果问题仍然存在，可以点击歌曲信息卡片上的「重试」按钮手动触发识别。

### Q: 可视化效果卡顿怎么办？
A: 请尝试：
1. 降低可视化灵敏度
2. 选择较简单的可视化模式（如 BARS）
3. 关闭不必要的浏览器标签页
4. 确保您的设备性能足够运行应用

### Q: 应用在移动设备上显示异常怎么办？
A: SonicVision 支持响应式设计，但在一些老旧或性能较低的移动设备上可能会出现显示异常。请尝试：
1. 使用最新版本的浏览器
2. 选择较简单的可视化模式
3. 关闭不必要的应用

### Q: 如何保存我的设置？
A: 应用会自动将您的设置保存到浏览器的本地存储中，下次访问时会自动加载这些设置。

## 故障排除

### 麦克风问题

**症状**：应用无法捕获音频，或音频质量差

**解决步骤**：
1. 检查麦克风是否正常工作
2. 确保麦克风未被其他应用占用
3. 尝试选择不同的音频输入设备
4. 重启浏览器

### 识别问题

**症状**：应用无法识别歌曲，或识别结果不准确

**解决步骤**：
1. 确保音乐音量足够大
2. 减少环境噪音
3. 尝试不同的音乐类型
4. 检查网络连接
5. 点击「重试」按钮手动触发识别

### 可视化问题

**症状**：可视化效果异常，或应用崩溃

**解决步骤**：
1. 刷新浏览器页面
2. 选择较简单的可视化模式
3. 调整可视化参数
4. 确保浏览器支持 WebGL（对于 3D 模式）
5. 尝试使用不同的浏览器

### 网络问题

**症状**：应用无法连接到 API 服务器，或识别过程缓慢

**解决步骤**：
1. 检查网络连接
2. 尝试使用有线网络连接
3. 减少网络流量
4. 稍后再试

## 性能优化

### 设备优化

- **使用现代设备**：SonicVision 在性能较好的设备上运行更流畅
- **保持设备冷却**：避免设备过热，这会影响性能
- **关闭不必要的应用**：减少后台应用对系统资源的占用

### 浏览器优化

- **使用最新版本**：确保使用最新版本的浏览器
- **启用硬件加速**：在浏览器设置中启用硬件加速
- **关闭扩展**：某些浏览器扩展可能会影响性能
- **使用 Chrome 或 Firefox**：这些浏览器对 WebGL 和音频 API 的支持较好

### 应用设置优化

- **选择适合的模式**：根据设备性能选择合适的可视化模式
- **调整参数**：降低灵敏度和速度可以提高性能
- **关闭不必要的功能**：如不需要歌词，可以关闭歌词显示
- **使用 2D 模式**：2D 模式通常比 3D 模式性能更好

### 网络优化

- **使用稳定的网络**：歌曲识别需要稳定的网络连接
- **减少网络延迟**：使用低延迟的网络连接
- **避免高峰时段**：在网络流量高峰时段，API 响应可能会变慢
